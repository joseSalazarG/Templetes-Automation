# Nombre del Proyecto

## Descripción

Este es un proyecto enfocado en la automatización de la página web "url". Utiliza una combinación de Selenium, Cucumber, JUnit y Maven junto con Java 21 para proporcionar una solución robusta y eficiente en la automatización de pruebas web.

***

1. **Lenguaje de Programación**
* Java (Versión 17)

* Selenium
* Cucumber
* JUnit

3. **Bibliotecas**

## Requisitos de Instalación

Para utilizar "Nombre del proyecto", es necesario tener instalado Java 17 y Maven. Las dependencias de las bibliotecas se gestionan a través de Maven, asegurándose de que todas las herramientas necesarias estén disponibles.

***

## Instrucciones de Uso

Para ejecutar las pruebas automatizadas en AdServer Web, sigue estos pasos:

1. Clona el repositorio a tu máquina local.
```
$ git clone <direccion del repo>
```
2. Abre el proyecto en tu IDE preferido.
3. Asegúrate de que Java 17 y Maven estén correctamente configurados en tu entorno de ejecución.
4. Para ejecutar las pruebas, utiliza el siguiente comando en tu terminal:
```
mvn test "-Dcucumber.filter.tags=@Nombre del proyecto"
```
Este comando ejecutará todas las pruebas etiquetadas con @Nombre del proyecto en Cucumber.

***

## Características del Proyecto

Integración completa con Selenium para pruebas de navegador.
Uso de Cucumber para pruebas BDD (Behavior Driven Development).
Generación de informes detallados con ExtentReports.
